================================================================================
HOW TO BUILD THE EXECUTABLE
================================================================================

Windows:
    1. Open Command Prompt
    2. Navigate to project root
    3. Run: build_tools\BUILD_ALL.bat
    4. Wait 2-5 minutes
    5. Find executable in: dist\ExcelDataProcessor.exe

That's it!

================================================================================
DEPLOYMENT STRUCTURE
================================================================================

After building, create this structure for distribution:

YourDeploymentFolder/
├── ExcelDataProcessor.exe    (from dist/)
